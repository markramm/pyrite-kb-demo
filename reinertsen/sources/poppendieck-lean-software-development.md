[[mary-poppendieck]] and [[tom-poppendieck]]'s foundational lean software development text. Cites Reinertsen extensively, particularly [[cost-of-delay]] and [[economic-framework-for-prioritization]], as the economic justification for lean practices in software. This book is a primary document of how Reinertsen's product development economics entered the software world, bridging [[taiichi-ohno]]'s manufacturing lean with software development practice.
