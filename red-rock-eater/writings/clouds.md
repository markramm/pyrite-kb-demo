```

Subject: Superb Software from Australia

One of the finest pieces of educational software there is comes from
The Curriculum Development and Ongoing Education Group of the
Australian Bureau of Meteorology

It is designed to teach Cloud Identification. It is written for Windows
using ToolBook by Ian Bell, Paddy Dillon and Peter Yew.

All the cloud types are represented. Besides learning how to identify
cloud types, users also find out about identifying them under nocturnal
conditions. Tests are built in, as well as information relating to aviation.

A student would come away from working happily with this visually
attractive, user-friendly program having learned a lot.

The program is so well designed that teachers who touch on weather in
their science programs would do well to obtain the program to have it
always available for students to work with. It is a program which does not
require much teacher supervision. In fact, with students working in pairs,
even testing and recording could be effectively carried out. Students could be
evaluated also according to how well after completing the program they
are able to identify real clouds.

With so much educational software little more than cosmetic and vapidly
entertaining, it is refreshing and exciting to discover software that
effectively uses a computer to help students learn.
```
