------------------------ Start of Message ------------------------

MCS-Alert (Mathematics and Computer Science - Alert) is a free
alerting, archiving and access service for the following
journals:

     Artificial Intelligence            (jrnl code: artint)
     Discrete Mathematics               (jrnl code: disc)
     Discrete Applied Mathematics       (jrnl code: dam)
     Theoretical Computer Science       (jrnl code: tcs)

providing rapid access to

     tables of contents of forthcoming and published issues
     abstracts of forthcoming and published papers
     full-text search
     index search

This service may be accessed in two ways:

1    gopher to gopher.elsevier.NL and select "Mathematics and
          Computer Science" (for details see the "About..."
          files)
2    by email to mcs-e@elsevier.NL (for details see below)

-------------------- ACCESS INSTRUCTIONS -------------------

For instructions to retrieve Tables of Contents, please refer to
the COMMON commands.

For instructions to retrieve Abstracts, please refer to the
ADVANCED commands.

-------------------- ACCESS RESTRICTIONS -------------------

Please note that the ADVANCED commands are ONLY for individuals
affiliated to institutes with a full subscription to the paper
journal.

-------------------- GENERAL INFORMATION -------------------

All requests should be sent to

     mcs-e@elsevier.NL

with the command in the Subject line of the message

Please note that the text <jrnl> should be substituted by the
appropriate journal code (either artint, disc, dam, or tcs)

Questions and comments should be sent

     To: mcs-e@elsevier.NL
     Subject: MESSAGE <your subject>

     <Your message>

-------------------- COMMON COMMANDS -----------------------

HELP                    send this file

HELP <jrnl>-c           send the file with COMMON commands for
                        journal <jrnl>

MESSAGE                 forward the body of my message to the
                        list owner (for questions and comments)

SUBSCRIBE <jrnl>-c      add my email address to the distribution
                        list for the tables of contents of
                        <jrnl>

UNSUBSCRIBE <jrnl>-c    remove my email address from the dis-
                        tribution list

GET <jrnl>-c/xxx/y[-z]  send tables of contents of <jrnl>, with
                        volume number xxx, issue[s] i [to z]
                        (Example:  get artint-c/70/1-2)

GET macros/tex/elsart   send LaTeX document styles for Elsevier
                        journals

FIND <jrnl>-c <keys>    search indexes of <jrnl> with search
                        parameters <keys>.  The indexes contain
                        authors, titles and publication details.
                        Multiple keywords and optional quali-
                        fiers are allowed; keys and qualifiers
                        (separated by spaces) can appear in any
                        order.

     word1 word2        select papers containing both word1 and
                        word2
     not word3          exclude papers containing word3

                        Optional qualifiers are

     forthcoming        forthcoming issues only
     year 199x          year 199x only
     before 199x        before year 199x only
     since 199x         since year 199x only
     all                all years

-------------------- ADVANCED COMMANDS ---------------------

These ADVANCED commands are only available for individuals
affiliated to institutes with a full subscription to the paper
journal

PERMIT                  request access permission to both
                        ADVANCED commands and Subscriber Section
                        of the MCS-Alert gopher server.  Please
                        state in the body of your message:

* organization or university* city* country* email address* journal

Notification will be sent as soon as
                        possible.  If you do not require access
                        to the Subscriber Section of the gopher
                        server, simply use the SUBSCRIBE command
                        (see below)

HELP <jrnl>-a           send the file with ADVANCED commands for
                        journal <jrnl>

MESSAGE                 forward the body of my message to the
                        list owner (for questions and comments)

SUBSCRIBE <jrnl>-a      add my email address to the distribution
                        list for the abstracts of <jrnl>

UNSUBSCRIBE <jrnl>-a    remove my email address from the dis-
                        tribution list

GET <jrnl>-a/xxx/y[-z]  send abstracts of <jrnl>, with volume
                        number xxx, issue[s] i [to z]
                        (Example:  get artint-a/70/1-2)

GET macros/tex/elsart   send LaTeX document styles for Elsevier
                        journals

SEARCH <jrnl>-a <keys>  full-text search of the abstracts of
                        <jrnl> with search parameters <keys>. 
                        Multiple keywords and optional quali-
                        fiers are allowed; keys and qualifiers
                        (separated by spaces) can appear in any
                        order.

     word1 word2        word1 OR word2: first, abstracts will be
                        listed that contain both word1 and
                        word2; next, abstracts will be listed
                        that contain word1 only and then the
                        abstracts with word2 only

     not word3          exclude abstracts containing word3

                        Optional qualifiers are:

     forthcoming        forthcoming issues only
     year 199x          year 199x only
     before 199x        before year 199x only
     since 199x         since year 199x only
     all                all years

------------------------- End of Message -------------------------
