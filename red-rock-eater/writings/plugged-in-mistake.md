# Plugged In mistake

```
It seems that the "Plugged In" message I sent to RRE yesterday "escaped" onto
the net by mistake.  Please forward this message from Nick Arnett anyplace you
forwarded the "Plugged In" message itself.

Date: Fri, 8 Jul 1994 09:29:40 -0800
From: nicka@mccmedia.com (Nick Arnett)

[...]

>Date:    Fri, 24 Jun 1994 11:43:00 -0700
>From:    "Arthur R. McGee" <amcgee@NETCOM.COM>
>Subject: Plugged In (fwd)
>
>National Youth Center Electronic Network Project Narrative

[Big sigh.]  This proposal is a draft that is no longer alive and was
never intended to be distributed.  I'm on the board of directors of Plugged
In and in charge of our strategic planning, so I'm familiar with the
situation.  This document was an exploratory proposal that Mario Morino
developed with Plugged In and the other organizations.  Mario isn't
particularly happy that it was put on the net and would like the
distribution to stop so that we can end the confusion that it is causing.

I'm not saying this to place any blame; we don't know who released a copy
originally.  It seems to have originally be posted on AOL a few weeks ago.

Please don't assume that this project is going forth as described.  Plugged
In, in particular, is narrowing its focus to the Bay Area and is developing
partnerships with other organizations to carry out many of the goals, so
that it can stay quite focused.

BTW, the executive director of Plugged In is named "Bart Decrem," please
note the correct spelling.  Bart's mailbox is filled with messages asking
about this project due to the dissemination of the report.  If you want
more information, we will be announcing a Plugged In Web server soon.  It
will contain information about our programs and plans.  It's in development
here at my site.

Nick

Multimedia Computing Corp.
Campbell, California

---

"We are surrounded by insurmountable opportunity." -- Pogo
```
