# RRE notes

```
As a periodic reminder, to end your subscription to RRE, send a message that
looks like:

  To: rre-request@weber.ucsd.edu
  Subject: unsubscribe

Another reminder: it is still my goal to get a copy of "Networking on the
Network" into the hands of every graduate student in the world.  If you happen
to know any graduate students who might not have seen this useful guide to the
art of professional networking, or you if you simply know somebody who knows 
a lot of graduate students, I would greatly appreciate it if you'd pass a copy
along to them.  To fetch it, send a message that looks like:

  To: rre-request@weber.ucsd.edu
  Subject: archive send network

An administrative note: my policy is to unsubscribe anybody whose computer
tells me "user unknown".  If you get unsubscribed without asking, that's
probably what happened.  Just go ahead and subscribe again, the same way you
did before, by sending a message that looks like:

  To: rre-request@weber.ucsd.edu
  Subject: subscribe yourfirstname yourlastname

Also, a while back I suggested that you subscribe to the Communet digest, a
very interesting discussion of community networking.  Unfortunately, I didn't
tell you have to get the digest, as opposed to getting every message in 
your box separately.  First, as I said, if you'd like to subscribe to the list
itself, send a message like:

  To: listserv@uvmvm.uvm.edu
  Subject: anything

  subscribe communet

Or, if you'd like to get the digest rather than a slew of separate messages,
send another message that looks like:

  To: listserv@uvmvm.uvm.edu
  Subject: anything

  subscribe communet
  set communet digest

(It's too bad the digest itself doesn't tell you this.)

Finally, I've enclosed descriptions of several files that have been added to
the RRE archive over the last month.  Most have been circulated on RRE, but
some have not.  To retrieve any of them, send a message that looks like this:

  To: rre-request@weber.ucsd.edu
  Subject: archive send <<filename>>

See you.

Phil
```
