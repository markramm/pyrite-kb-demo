# technical difficulties

```
In the last week several people have told me that they've been getting RRE
messages without asking for them.  If you're in this situation, you can end
your subscription by sending a message like so:

  To: rre-request@weber.ucsd.edu
  Subject: unsubscribe

At the moment we have no idea why this might be happening.  If you're getting
RRE messages you don't want, could you send us (rre-maintainers@weber.ucsd.edu)
a message?  Tell us, if possible, the entire history of your dealings with our
server (the one whose address is rre-request@weber.ucsd.edu), so we can have
some clue about what the problem might be.

Thanks a lot.

Phil

PS I'm sorry for all of these administrative notes.
```
