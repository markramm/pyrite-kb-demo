# [RRE]new RRE server

```
The Red Rock Eater News Service has moved to a new server at UCLA.
We are using a list of addresses that we captured from the UCSD server
about ten days ago, and so if you have unsubscribed recently then you
may have to unsubscribe again at the new address.  Just send a message
that looks like this:

  To: requests@lists.gseis.ucla.edu
  Subject: unsubscribe rre

The new server presumably has some bugs, so I hope that you will bear
with us as we work through them.

Here is my new contact information:

  Graduate School of Education and Information Studies

  USA

  phone: +1 (310) 825-7154 (my office phone)

    fax: +1 (310) 206-4460
    net: pagre@ucla.edu
   home: http://dlis.gseis.ucla.edu/pagre/

Starting next week, Paul Duguid <duguid@socrates.berkeley.edu> will be
taking over, probably for a couple of months, as the guest editor of
RRE.  The list is approaching its fifth anniversary and I need a rest.
Paul is a good guy, I hope that you will support him by sending him lots
of good stuff for the list.  I will probably be contributing items to
the list myself, assuming that Paul sees fit to send them out.  You will
recall that I announced that Paul was taking over the list last summer,
but that never ended up working out technically.  I hope it'll work this
time.

Have a good summer.

Phil
```
